HoneyMap

You are looking at the Modern Honeynet HoneyMap, a real-time world map which visualizes attacks Red markers on the map represent attackers, yellow markers are targets (honeypot sensors).

This project is experimental and should be considered a BETA version. So far, current Chrome and Firefox browsers should work fine. Opera, Safari and Internet Explorer probably wont work completely. If you identify bugs or have feature requests, please let us know.
